Syntax extensions and various conveniences are offered for a variety of types.

The extensions are organized by source. In particular:


- Cats
  - Effect
  - IO
- Scala
  - Any
  - Future
  - Iterable
  - Long
  - String
  - Try
- Scalacheck
- Twitter
  - Try
